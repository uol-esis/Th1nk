# Th1nk

Th1nk is a project which aims to simplify the process of extracting and visualizing data from messy Excel files.

## Installation

To start Th1nk, simply run the following command:

```bash
./install.sh
```

> [!CAUTION]
> This will start Th1nk in an unsecure way. Please consult the wiki for a properly secured deployment.
