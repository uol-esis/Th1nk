# Th1nk

Th1nk is a project which aims to simplify the process of extracting and visualizing data from messy Excel files.

## Deployment

Th1nk provides a deployment script which downloads all required files to start the application.

### Prerequisites

- Docker

```bash
curl -o deploy.sh https://raw.githubusercontent.com/uol-esis/th1nk/main/deployment/deploy.sh
chmod +x deploy.sh
./deploy.sh
```

Now follow the instructions in the terminal to complete the setup.
